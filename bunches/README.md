#### TL/DR

- `!` - quit app
- `!!!` - run on close
- `%` - ignore on close
- `~5` (at the end) - delay start
- `@` - Focus on app
- `@@` - hide everything
- `x-bunch://close/<bunchname>` / `x-bunch://open/<bunchname>` - to open/close other bunches
